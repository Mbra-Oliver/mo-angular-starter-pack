Mbra Oliver - Angular Starter Pack

Un starter Angular 20+ moderne, propre et scalable pour débuter rapidement vos projets.

🚀 Fonctionnalités clés

  • Structure modulaire claire (UI, core, etc.)
  • Composants UI réutilisables
  • Intercepteurs HTTP configurés
  • Utilitaires, interfaces, guards, resolvers
  • Prêt à intégrer Tailwind CSS

⚙️ Installation

npm install
ng serve

Accédez à l'application sur http://localhost:4200/

📦 Structure du projet

src/
├── app/
├── components/ui/        # UI Components
├── core/
│   ├── guards/
│   ├── interceptors/
│   ├── interfaces/
│   ├── resolvers/
│   ├── utils/
│   └── utils-components/
